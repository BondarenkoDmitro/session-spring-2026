# Daily Study Planner

Веб-приложение для планирования учебных задач по дням и неделям.

## Что умеет сейчас
- Переключение по неделям с подсветкой активной кнопки.
- Карточки дней с задачами и прогрессом.
- Статусы задачи: `Не начато`, `В работе`, `Сделано`.
- Прогресс-бар от `0%` до `100%` (шаг 5%).
- Модальные окна по клику на статистику: `Всего`, `Выполнено`, `В работе`, `Не начато`.
- Добавление задачи через отдельное модальное окно (а не в карточке дня).
- На кнопке `Сделано` запускается короткий таймер и затем запрос на удаление задачи.
- Если после удаления задач день пустой, можно удалить и сам день.
- Автосохранение в `localStorage`.

## Текущий поток добавления задачи
1. Нажмите `➕ Добавить задачу` в карточке дня
   или кнопку `Сделать задачу` в модалке `Всего задач`.
2. Откроется отдельное модальное окно создания задачи.
3. Заполните поля:
   - Описание
   - Дедлайн (дата)
   - Время дедлайна
   - Время на работу (часы)
   - Предмет
4. Нажмите `Сделать задачу`.

## Запуск проекта
Требования:
- Node.js 18+
- npm

```bash
cd session-spring
npm install
npm run dev
```

Откройте в браузере адрес из Vite (обычно `http://localhost:5173`).

## Сборка
```bash
cd session-spring
npm run build
npm run preview
```

## Хранилище данных
- Данные сохраняются в браузере (`localStorage`, ключ `plannerData_v2`).
- Если очистить `localStorage`, прогресс и изменения сбросятся.

## Структура проекта
- `session-spring/index.html` — корневая разметка.
- `session-spring/src/main.js` — логика приложения.
- `session-spring/src/style.css` — стили.
