# 📋 Ежедневный План Обучения

Веб-приложение для управления вашим расписанием и задачами. Помогает отслеживать прогресс обучения с красивым интерфейсом и удобными инструментами.

---

## 🎯 Основные возможности

✅ **Управление задачами** - добавляйте, удаляйте и отслеживайте задачи  
✅ **Три статуса** - Сделано / В работе / Не начато  
✅ **Ползунок прогресса** - отмечайте прогресс от 0% до 100%  
✅ **Выбор по неделям** - легко переключайтесь между неделями  
✅ **Автосохранение** - все данные сохраняются в браузер  
✅ **Красивый дизайн** - черно-синяя цветовая схема с анимациями

## 📖 Как пользоваться

### 1️⃣ Выбор недели
- В верхней части страницы нажимайте кнопки **"Неделя 1"**, **"Неделя 2"** и т.д.
- Текущая неделя выделена синим цветом
- На странице отобразятся все дни выбранной недели

### 2️⃣ Просмотр дня и задач
- Каждый день представлен отдельной карточкой
- В карточке видны все задачи на этот день
- Каждая задача имеет дату и время (4-5 часов)

### 3️⃣ Отмечание статуса задачи
Нажимайте одну из трёх кнопок:

- **✅ Сделано** - задача полностью выполнена (прогресс 100%)
- **⏳ В работе** - вы работаете над задачей (прогресс ~50%)
- **❌ Не начато** - задача еще не начата (прогресс 0%)

### 4️⃣ Регулировка прогресса
- Под каждой задачей есть **ползунок прогресса** (0-100%)
- **Кликните на полосу** в нужном месте, чтобы установить процент
- Процент автоматически выравнивается на 5% (0%, 5%, 10%... 100%)

### 5️⃣ Добавление новой задачи
- В каждом дне внизу есть поле **"Описание новой задачи"**
- Введите текст задачи
- Нажмите **➕ Добавить**
- Новая задача появится в списке

### 6️⃣ Удаление
- **Удалить задачу** - нажмите 🗑️ рядом с задачей
- **Удалить весь день** - нажмите 🗑️ "Удалить" вверху карточки дня

### 7️⃣ Статистика
В верхней части видна статистика:
- **Всего задач** - общее количество
- **✅ Выполнено** - сколько сделано
- **⏳ В работе** - сколько в процессе
- **❌ Не начато** - сколько еще не начинали

## 💾 Сохранение данных

- ✅ Все ваши изменения **автоматически сохраняются** в браузер
- ✅ Даже если вы закроете браузер, данные останутся
- ✅ Кеш хранится в **localStorage** браузера
- ⚠️ **Очистка кеша браузера удалит все данные!**
